## ✨ Key Features

- **🧠 LLM Policy Translation:** Type any natural language policy (e.g., *"Make public transport free"*). The AI parses it into mathematical simulation parameters.
- **📊 RAG Data Grounding:** The simulation doesn't run on random numbers. It pulls real-world city datasets (e.g., Bhubaneswar 2026 dataset) to ground the AI's reasoning.
- **🏙️ Agent-Based Spatial Modeling:** Watch autonomous AI citizens (agents) move, react, and change behavior on a 3D MapLibre/Deck.gl spatial grid.
- **⚡ Real-Time WebSockets:** Live streaming of simulation steps, updating pollution and traffic density charts with zero lag.
- **🗣️ Live Citizen Personas:** A live "Twitter-style" feed where generated AI personas (Taxi Driver, Student, etc.) react to the policy as the simulation runs.
- **📄 Automated AI Reporting:** Once the simulation completes, a Chief Data Scientist AI Agent automatically writes a professional Executive Impact PDF Report.

## 🛠️ Tech Stack

### Frontend (User Interface & 3D Mapping)
- **Framework:** Next.js 14, React
- **Styling:** Tailwind CSS, Lucide React (Icons)
- **Data Visualization:** Recharts (Live Graphs)
- **Spatial Rendering:** Deck.gl, MapLibre GL, Carto Dark Matter (Free Map Layer)

### Backend (Simulation & Streaming)
- **Framework:** FastAPI, Python, Uvicorn
- **Real-Time Communication:** WebSockets
- **Simulation Engine:** Custom Python Agent-Based Continuous Space Engine

### Artificial Intelligence
- **LLM:** Google Gemini 1.5 Flash
- **Agent Orchestration:** CrewAI, LangChain
- **Context Handling:** Lightweight RAG (Retrieval-Augmented Generation)

## ⚙️ Setup Instructions

Follow these steps to run the FutureTwin Engine locally on your machine.

### Prerequisites
- Node.js (v18+)
- Python (3.10+)
- A [Google Gemini API Key](https://aistudio.google.com/app/apikey)

### 1. Clone the Repository
```bash
git clone [https://github.com/yourusername/ai-futuretwin.git](https://github.com/yourusername/ai-futuretwin.git)
cd ai-futuretwin

### Backend Setup (FastAPI & AI Agents)

# Navigate to the backend directory
cd backend

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate

# Install required Python packages
pip install fastapi uvicorn crewai langchain-google-genai python-dotenv

# Set up environment variables
# Create a .env file inside the backend folder and add your Gemini API Key
echo "GEMINI_API_KEY=your_actual_api_key_here" > .env

# Start the WebSocket Server
python -m uvicorn app.main:app --reload

The backend will start running at ws://127.0.0.1:8000/ws/simulate

3. Frontend Setup (Next.js)

Open a new terminal window and navigate to the frontend folder.

# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Install mapping and charting libraries
npm install react-map-gl maplibre-gl @deck.gl/react @deck.gl/layers recharts lucide-react

# Start the Next.js development server
npm run dev

The frontend will start running at http://localhost:3000
